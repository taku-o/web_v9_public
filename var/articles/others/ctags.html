.. |yen| unicode:: U+000A5 .. YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ Ctagsは、ソースコード内の変数、メソッドが宣言されている個所を探し出し、 その情報を持ったタグファイルを作るプログラムです。 |BR| 作成されたタグファイルは他のアプリケーションが利用します。 ================================================================================ インストール方法 (Win) ================================================================================ まず、ctagsの日本語版を入手します。 `h_east's website`__ __ http://hp.vector.co.jp/authors/VA025040/ctags/ 圧縮ファイルを解凍したら、解凍フォルダ内の「ctags.exe」をPATHの通るディレクトリに置きます。 ================================================================================ インストール方法 (Mac) ================================================================================ MacOSX 10.3以降であれば、 ctagsコマンドはインストールされています。 .. note:: 下記は日本語版パッチありのctagsをMacOSXにインストールする場合の 古いドキュメントです。 `Exuberant Ctags`_ と `Exuberant CtagsのUNIX版日本語パッチ`_ を入手します。 .. _`Exuberant Ctags`: http://ctags.sourceforge.net/ .. _`Exuberant CtagsのUNIX版日本語パッチ`: http://hp.vector.co.jp/authors/VA025040/ctags/ 各圧縮ファイルを解凍したら、 |BR| Exuberant Ctagsディレクトリに、日本語版パッチをコピーします。 次にExuberant Ctags解凍ディレクトリに移動して、 下のコマンドでコンパイルできます。(ctags5.3J1の場合) .. parsed-literal:: :class: console patch -c < ctags-5.3-j1.patch ./configure make コンパイル後は、 .. parsed-literal:: :class: console make install でインストールするか、 |BR| PATHを通せばコマンドが使用可能になります。 ================================================================================ 使用方法 ================================================================================ タグファイルを生成したいソースコードのあるディレクトリに移動して、 .. parsed-literal:: :class: console ctags -R と入力するとタグファイルが作成されます。 ================================================================================ 注意事項など ================================================================================ 最終的にはタグファイルを役立てられる、別のアプリケーションが必要となります。 ================================================================================ 使用方法 (vimエディタ編) ================================================================================ まず、タグファイルを作成してください。 |BR| メニューの「ツール(T)」から「タグファイル作成(T)」を選ぶか、 |BR| Exモードで下のコマンドを実行します。 .. parsed-literal:: :class: console !ctags -R タグファイルを作成したら、ノーマルモードでメソッドや、変数上にカーソルを移動し、 .. parsed-literal:: :class: console Ctr-] を実行すると、変数宣言、メソッド宣言のある行に移動できます。 それがファイル名であれば、そのファイルを開くことができます。 (カレントディレクトリのみです。) タグジャンプは、同一ファイル内にメソッド、変数宣言がなされている場合のみ有効です。
