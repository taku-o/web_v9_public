YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ Phrases : Find and highlight words／phrases used repeatedly |BR| http://www.vim.org/scripts/script.php?script_id=1005 Visualモードで選択した範囲をハイライトします。 |BR| ハイライトされた行は、Visualモードで次のハイライト箇所が指定されるまで、 継続してハイライトされ続けます。 ================================================================================ インストール方法 ================================================================================ `phrases.vimのスクリプト`__ をダウンロードしたら、 |BR| ダウンロードしたスクリプトをpluginディレクトリにコピーしてください。 __ http://www.vim.org/scripts/script.php?script_id=1005 .. csv-table:: :header: "ファイル", "URL" "phrases.vim", "http://www.vim.org/scripts/script.php?script_id=1005" ================================================================================ 使用方法 ================================================================================ Visualモードで範囲を選択します。 .. image:: /dist/img/vim/phrases_visual.jpg :alt: 範囲選択 「Enterキー」を押してください。 |BR| このスクリプトの効果で、選択行がハイライトされます。 .. image:: /dist/img/vim/phrases_hilight.jpg :alt: ハイライト vimエディタのカーソルを移動しても、次にVisualモードでの選択を行って 「Enterキー」を押さない限り、 ハイライトは継続されます。 ================================================================================ 注意事項など ================================================================================ このスクリプトを使用すると、 Visualモードの挙動が、デフォルトとは少し変わってしまうので、 気をつけてください。
