YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ `GENE辞書`__ による 英和辞書検索を行うことのできるスクリプトです。 |BR| Kaoriya版vimに添付しています。 __ http://www-nagao.kuee.kyoto-u.ac.jp/member/tsuchiya/sdic/data/gene.html .. image:: /dist/img/vim/dicwin_focus.jpg :alt: GENE辞書による英単語検索結果表示 ================================================================================ インストール方法 ================================================================================ `GENE辞書`__ をダウンロードします。 __ http://www-nagao.kuee.kyoto-u.ac.jp/member/tsuchiya/sdic/data/gene.html GENE辞書は圧縮ファイルの形式で配布されているので、 これを解凍し、取り出した「gene.txt」ファイルを .. parsed-literal:: :class: program **Windowsの場合** $HOME/vimfiles/ または $HOME/vimfiles/dict/ .. parsed-literal:: :class: program **Mac OSXの場合** $HOME/.vim/ または $HOME/.vim/dict/ にコピーしてください。 |BR| (もし、ディレクトリが無い場合はディレクトリを作成すれば良い) .. csv-table:: :header: "ファイル", "URL" "GENE辞書", "http://www-nagao.kuee.kyoto-u.ac.jp/member/tsuchiya/sdic/data/gene.html" ================================================================================ 使用方法 ================================================================================ 英単語の辞書検索 ------------------------------------------------------------------------------- 調べたい対象の英単語の上にカーソルを持っていき、 |BR| Ctrlキーを押しながら「k」キーを押し、 |BR| もう一度「k」キーを押すと検索を行います。 .. parsed-literal:: :class: console **検索を行う** k 検索を行うと、検索結果表示用のミニウィンドウが開かれます。 .. image:: /dist/img/vim/dicwin.jpg :alt: GENE辞書による英単語検索 検索結果候補の表示 ------------------------------------------------------------------------------- 検索結果表示ウィンドウが開いている状態で、Ctrlキーを押しながら「k」キーを押した後、 - 次に「n」キーを押すと次の検索結果候補を - 次に「p」キーを押すと1つ前の検索結果候補を 表示します。 .. parsed-literal:: :class: console **次の候補を表示する** n .. parsed-literal:: :class: console **前の候補を表示する** p 検索結果表示ウィンドウにカーソルを移動する ------------------------------------------------------------------------------- 検索結果表示ウィンドウが開いている状態で、Ctrlキーを押しながら「k」キーを押した後、 |BR| 「w」キーを押すと、検索結果表示ウィンドウにカーソルが移動します。 |BR| .. parsed-literal:: :class: console **検索結果表示ウィンドウにカーソルを移動する** w カーソルが検索結果表示ウィンドウに移動すると、 検索結果表示ウィンドウの表示サイズが大きくなって、 検索結果が見やすくなります。 .. image:: /dist/img/vim/dicwin_window.jpg :alt: 検索結果表示ウィンドウが大きくなる 検索結果表示ウィンドウを閉じる ------------------------------------------------------------------------------- 検索結果表示ウィンドウが開いている状態で、Ctrlキーを押しながら「k」キーを押した後、 |BR| 「c」キーを押すと、検索結果表示ウィンドウを閉じることができます。 .. parsed-literal:: :class: console **検索結果表示ウィンドウを閉じる** c 検索単語を直に指定して検索を行う ------------------------------------------------------------------------------- 辞書検索は、英単語の上で |BR| k |BR| を実行する方法以外に、 .. parsed-literal:: :class: console **辞書検索する単語の入力** / を実行することにより、 |BR| 英単語を直に入力して検索するような方法をとることもできます。 .. image:: /dist/img/vim/dicwin_input.jpg :alt: 入力ワードによる辞書検索
