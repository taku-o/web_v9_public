YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ taglist.vim : Source code browser (supports C, C++, java, perl, python, tcl, sql, php, etc) |BR| http://www.vim.org/script.php?script_id=273 `Exuberant Ctags`__ で生成したタグファイルより、 別ウィンドウに変数、メソッド一覧を表示するスクリプトです。 __ http://ctags.sourceforge.net/ ================================================================================ インストール方法 ================================================================================ Exuberant Ctagsをインストールします。 |BR| 次に taglist.vim を$VIMRUNTIME/plugin にコピーします。 次に_vimrc、もしくは.vimrcファイルに .. parsed-literal:: :class: program :set tags=tags .. csv-table:: :header: "ファイル", "URL" Exuberant Ctags, http://ctags.sourceforge.net/ taglist.vim, http://www.vim.org/script.php?script_id=273 ================================================================================ 使用方法 ================================================================================ まず、タグファイルを生成します。 次に .. parsed-literal:: :class: console :Tlist で、別ウィンドウにファイル内の変数宣言、メソッド宣言のリストが表示されます。 .. image:: /dist/img/vim/taglist.jpg タグリストのウィンドウに移動して、 移動したい変数、メソッド上でEnterキーを押すと、 その宣言個所に移動します。 また、現在開いているファイルを変更すると、 別ウィンドウのリストも、それに合わせて、切り替わります。 ================================================================================ 注意事項など ================================================================================ このスクリプトは、 `minibufexpl.vimスクリプト`__ と相性がとても悪いようです。 __ /articles/vim/minibufexpl_vim.html
