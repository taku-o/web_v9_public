================================================================================ colorsel.vim : A RGB/HSV color selector |BR| http://www.vim.org/scripts/script.php?script_id=927 colorsel.vimプラグインは、Gimp2のRGB/HSV Color Selectorのようなユーザーインターフェイスで、 |BR| 色を選択させるプラグインです。選択した色はRGBのコードで取得できます。 |BR| このプラグインは、Web系の開発や、画像編集ソフトのお供として、利用できるでしょう。 .. image:: /dist/img/vim/colorsel_panel_open.png :alt: カラーセレクタ表示 ================================================================================ インストール方法 ================================================================================ vimオフィシャルサイトから `colorsel.vimプラグイン`__ をダウンロードして、 vim設定ファイルディレクトリのpluginディレクトリにコピーしてください。 |BR| （pluginディレクトリが無い場合は作成します。） __ http://www.vim.org/scripts/script.php?script_id=927 .. csv-table:: :header: ファイル, URL colorsel.vim, http://www.vim.org/scripts/script.php?script_id=927 ================================================================================ 使用方 ================================================================================ カラーセレクタを表示する -------------------------------------------------------------------------------- 「:ColorSel」コマンドを実行すると、色選択用のウィンドウが開かれます。 .. parsed-literal:: :class: console :ColorSel .. image:: /dist/img/vim/colorsel_command.png :alt: 「:ColorSel」コマンド実行 ↓ 開かれたウィンドウ .. image:: /dist/img/vim/colorsel_panel_open.png :alt: カラーセレクタ表示 カラーセレクタを操作する -------------------------------------------------------------------------------- 開かれたcolorsel.vimのウィンドウでは、普段、Vimエディタのカーソル移動で使用している、 「j」「k」「h」「l」、他のキーで色を選択できます。 .. image:: /dist/img/vim/colorsel_panel.png :alt: カラー選択 .. csv-table:: :header: キー, 説明 "j, k", 上下に移動する。 "h, l, w, b, 0, $", 左右に移動する。 "q", ウィンドウを閉じる。 "y", 色をRGBのコードで取得して、無名レジスタにセットします。 RGBのコードを貼り付ける -------------------------------------------------------------------------------- 色を選択した後は「y」キーを押すと、選択した色のコードが無名レジスタにコピーされるので、 .. image:: /dist/img/vim/colorsel_color_selected.png :alt: 色選択、ヤンク 通常のテキスト操作通り、「p」キーでペーストできます。 .. image:: /dist/img/vim/colorsel_paste.png :alt: 色選択、ヤンク ================================================================================ 注意事項など ================================================================================ 今回紹介したcolorsel.vimプラグインは、GUI版のVimエディタでのみ利用できます。
