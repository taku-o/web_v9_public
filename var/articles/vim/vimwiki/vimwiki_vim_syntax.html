================================================================================ テーブル ================================================================================ テーブル構造は各セルを「||」で区切って、値を並べることで表現します。 .. parsed-literal:: :class: program \|\|Years\|\|Temperature (low)\|\|Temperature (high)\|\| \|\|1900 \|\|-10 \|\|25 \|\| \|\|1910 \|\|-15 \|\|30 \|\| \|\|1920 \|\|-10 \|\|32 \|\| \|\|1930 \|\|-20 \|\|29 \|\| \|\|1940 \|\|-2 \|\|40 \|\| .. raw:: html 
    

    

    YearsTemperature (low)Temperature (high)
    

    

    1900 -10              25                
    

    

    1910 -15              30                
    

    

    1920 -10              32                
    

    

    1930 -20              29                
    

    

    1940 -2               40
テーブルのセルの中でも、リンクやボールド書体を扱えます。 |BR| 左寄せ、右寄せ、センタリングは、セルの左端右端、どちらに半角スペースがあるか、両方にあるか、でコントロールします。 .. parsed-literal:: :class: program \|\| Sequence \|\| セル内のコントロール \|\| 文字装飾、リンク \|\| \|\|090518001 \|\|左寄せ \|\| \*ボールド\* \|\| \|\|090518002 \|\| 右寄せ\|\| _イタリック_ \|\| \|\|090518003 \|\| センタリング \|\| ``http://nanasi.jp/`` \|\| ================================================================================ リスト ================================================================================ 通常リスト -------------------------------------------------------------------------------- 行の先頭を「空白文字」と「*」で始めると、その行はリストとして表現されます。 .. parsed-literal:: :class: program * Bulleted list item 1 * Bulleted list item 2 * Bulleted list sub item 1 * Bulleted list sub item 2 * more ... * and more ... * ... * Bulleted list sub item 3 * etc. .. parsed-literal:: :class: program
  • Bulleted list item 1
  • Bulleted list item 2
    • Bulleted list sub item 1
    • Bulleted list sub item 2
    • more ...
      • and more ...
      • ...
    • Bulleted list sub item 3
    • etc.
数字付きリスト -------------------------------------------------------------------------------- 行の先頭を「空白文字」と「*」ではなく、「空白文字」と「#」にすると、 番号付きのリストになります。 .. parsed-literal:: :class: program # Numbered list item 1 # Numbered list item 2 # Numbered list sub item 1 # Numbered list sub item 2 # more ... # and more ... # ... # Numbered list sub item 3 # etc. .. parsed-literal:: :class: program
  1. Numbered list item 1
  2. Numbered list item 2
    1. Numbered list sub item 1
    2. Numbered list sub item 2
    3. more ...
      1. and more ...
      2. ...
    4. Numbered list sub item 3
    5. etc.
================================================================================ TODOリスト ================================================================================ リストの先頭に「\[ ]」を入れると、そのリスト項目はTODOリストとして扱われます。 .. parsed-literal:: :class: program * \[ ] Bulleted list item 1 * \[ ] Bulleted list item 2 * \[ ] Bulleted list sub item 1 * \[ ] Bulleted list sub item 2 * \[ ] more ... * \[ ] and more ... * \[ ] ... * \[ ] Bulleted list sub item 3 * \[ ] etc. TODOリストの項目の上では、Controlキーを押しながらSpaceキーを押下することで、 |BR| \[ ] の中にチェックを入れたり、外したりできます。 .. raw:: html 
    * [x] Bulleted list item 1
    * [ ] Bulleted list item 2
       * [ ] Bulleted list sub item 1
.. image:: /dist/img/vim/vimwiki_todolist_with_indent.png :alt: TODOリスト画像 :class: image-border ================================================================================ 内部リンク ================================================================================ Wikiページ間でリンクを張るには、 英語大文字始まりのキャメルケースの綴りの語を書くか、 .. parsed-literal:: :class: program LinkToTopPage 角括弧２つでリンクを張りたい語を囲みます。 .. parsed-literal:: :class: program \[\[link to Top Page]] .. image:: /dist/img/vim/vimwiki_pre_link.png :alt: リンク前画像 ↑↓ リンクは、リンク先のページが無いと、Wikiページ上でハイライト表示されます。 .. image:: /dist/img/vim/vimwiki_post_link.png :alt: リンク後画像 ================================================================================ 外部リンク ================================================================================ テキストリンク -------------------------------------------------------------------------------- Vimwiki外部のドキュメントに対してハイパーリンクを張るには、 |BR| URLを直接記述するか、各括弧でURLを囲みます。 .. parsed-literal:: :class: program ``http://nanasi.jp/var/articles/vim/vimwiki/vimwiki_vim.html`` \[``http://nanasi.jp/``] \[``http://nanasi.jp/`` 名無しのvim使い] \[\[``./index.html``\|名無しのvim使い]] 画像リンク -------------------------------------------------------------------------------- VimwikiのWikiページをHTML化した際に、画像を表示する方法は、テキストリンクを張る方法と似ています。 |BR| Wikiページに表示したい画像のURLを書くか、角括弧2つでURLを囲みます。 .. parsed-literal:: :class: program ``http://nanasi.jp/logo.png`` \[\[``http://nanasi.jp/logo.png``]] \[\[``image/logo.png``]] \[\[``http://nanasi.jp/logo.png``\|http://nanasi.jp/log_thumbnail.png]] \[\[``http://nanasi.jp/logo.png``\||width:150px; height:120px;]] \[\[``http://nanasi.jp/logo.png``\|http://nanasi.jp/log_thumbnail.png\|width:150px; height:120px;]] Vimwikiでサポートされるのは、 拡張子が「.png」「.jpg」「.gif」「.jpeg」の画像です。 ================================================================================ 書体 ================================================================================ Vimwikiで、書体を扱うには次のように記述します。 .. parsed-literal:: :class: program \*ボールド\* _イタリック_ ~~取り消し線~~ \`書体なし（コード記述用）\` 本文^上付文字^ 本文,,下付文字,, HTML化した際には、それぞれ次のように表示されます。 .. raw:: html

================================================================================ パラグラフ ================================================================================ 行頭が空白文字から始まらない場合、その行をパラグラフとして扱います。 |BR| 例外は整形済みテキスト、テーブルの場合です。 ================================================================================ 整形済みテキスト ================================================================================ 行頭を空白文字で始めるか、指定範囲を「{{{」「}}}」で囲むと、 |BR| それらテキストを整形済みテキストとして扱います。 .. parsed-literal:: :class: program {{{ 行頭を空白文字で始めるか、指定範囲を「{{{」「}}}」で囲むと、 それらテキストは整形済みテキストとして扱います。 }}} HTML化すると、次のようなHTMLソースが生成されます。 .. parsed-literal:: :class: program 
    行頭を空白文字で始めるか、指定範囲を「{{{」「}}}」で囲むと、
    それらテキストは整形済みテキストとして扱います。
整形済みテキストのスタイル指定 -------------------------------------------------------------------------------- 整形済みテキストには属性を設定できます。 主にスタイルシートのためのクラスやIDを設定することになると思います。 .. raw:: html 
    {{{class="pre_class" id="pre_txt"
    行頭を空白文字で始めるか、指定範囲を「{{{」「}}}」で囲むと、
    それらテキストは整形済みテキストとして扱います。
    }}}
