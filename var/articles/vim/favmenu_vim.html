YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ FavMenu.vim : A new menu, like IE's Favourites, or Netscape's Bookmark |BR| http://www.vim.org/scripts/script.php?script_id=161 `favex.vim`__ と同じ機能を持つ、ファイル管理用ユーティリティです。 編集機会の多いファイルをFavourite ( いわゆるお気に入り ) として登録しておき、 そのファイルを編集したいときに、素早く開くことを可能にするスクリプトです。 __ /articles/vim/favex.vim favex.vimと違い、このスクリプトはメニューバーでファイルを管理します。 そのため、ファイルの選択、管理を容易に行うことが出来ます。 .. image:: /dist/img/vim/favmenu.jpg :alt: メニューバー 反面、メニューバーでお気に入りを管理する以上、 メニューバーを表示しない状況では、このスクリプトは役に立ちません。 |BR| Mac OSX版gvimでは、メニューバーを隠すことが出来ませんから、 それなりには使えるスクリプトとなることでしょう。 ================================================================================ インストール方法 ================================================================================ $VIMRUNTIME/pluginディレクトリに、入手したスクリプトをコピーして下さい。 .. csv-table:: :header: "ファイル", "URL" "FavMenu.vim", "http://www.vim.org/scripts/script.php?script_id=161" ================================================================================ 使用方法 ================================================================================ Favouritesメニューには、4種類の項目が用意されています。 .. csv-table:: :header: "項目名", "効果" "Add current file", "現在、編集中のバッファのファイルをお気に入りに登録します。" "Remove current file", "現在編集中のバッファのファイルがお気に入りにある時、そのファイルをお気に入りから取り除きます。" "Edit favourites", "お気に入りリストを直に編集します。主に削除するときに使用することになるでしょう。" "Refresh", "メニューを描画し直します。この項目を選んだ直後は、お気に入りに登録したファイルが表示されなくなりますが、お気に入りのリストが消滅するわけではないので安心して下さい。" ================================================================================ 注意事項など ================================================================================ gvimでメニューバーを表示するには、 .. parsed-literal:: :class: console :set guioption+=m メニューを隠すには、 .. parsed-literal:: :class: console :set guioption-=m を_gvimrc設定ファイルに書き込みます。
