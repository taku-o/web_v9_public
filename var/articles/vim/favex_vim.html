================================================================================ `vim online`__ から、配布ファイルを ダウンロードして、zipファイルを入手してください。 __ http://www.vim.org/script.php?script_id=539 .. csv-table:: :header: "ファイル", "URL" "FavEx.vim", "http://www.vim.org/script.php?script_id=539" zipファイルの中身は、プラグイン(favex.vim)、ドキュメント(favex.txt)、リスト管理用のファイル(favlist)です。 |BR| プラグインは$VIMRUNTIME/pluginに、 |BR| ドキュメントは$VIMRUNTIME/docに、 |BR| リスト管理用ファイルは$VIMRUNTIMEに |BR| それぞれコピーします。 最後にFavExのドキュメントを有効にします。vimを起動して、次のコマンドを入力してください。 .. parsed-literal:: :class: console :helptags $VIMRUNTIME/doc これで、準備は終了です。 ================================================================================ 使用方法 ================================================================================ $VIMRUNTIMEに置いたfavlistというファイルがリスト管理用ファイルです。 |BR| このファイルに、任意のファイル、ディレクトリを登録します。 リスト上の開きたいファイル、ディレクトリの上でEnterキーを押すことで、 そのファイル、ディレクトリをvimで開くことができます。 ファイル、ディレクトリ登録 ------------------------------------------------------------------------------- ファイルを開いてノーマルモード時に 下記のコマンドで、そのファイルをfavlistのリストに登録できます。 .. parsed-literal:: :class: console ff 次のコマンドは、そのファイルの親ディレクトリを favlistのリストに登録します。 .. parsed-literal:: :class: console fd リストを開く ------------------------------------------------------------------------------- ノーマルモード時に次のコマンドを入力することで、favlistを開けます。 .. parsed-literal:: :class: console fe .. image:: /dist/img/vim/favex_list.jpg :alt: favlist 次のコマンドは、ウィンドウを分割して、そこにfavlistを開くコマンドです。 .. parsed-literal:: :class: console fs .. image:: /dist/img/vim/favex_split.jpg :alt: fsの場合 favlistリスト上の操作 ------------------------------------------------------------------------------- 下の画像は、favlistを表示した状態です。 .. image:: /dist/img/vim/favex_list.jpg
