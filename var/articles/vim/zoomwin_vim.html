YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ ZoomWin : Brief-like ability to zoom into／out-of a window |BR| http://www.vim.org/scripts/script.php?script_id=508 複数のバッファを同時に編集する際に、 |BR| 複数のウィンドウで複数のバッファを表示、 |BR| 単一のウィンドウで特定のバッファを表示、 |BR| と、コマンドひとつで切り替えられるようにするプラグインです。 ================================================================================ インストール方法 ================================================================================ `vim online`__ から、スクリプトファイルをダウンロードしたら、 |BR| $VIMRUNTIME/pluginなどのプラグインディレクトリにコピーしてください。 __ http://www.vim.org/scripts/script.php?script_id=508 .. csv-table:: :header: ファイル, URL ZoomWin.vim, http://www.vim.org/scripts/script.php?script_id=508 ================================================================================ 使用方法 ================================================================================ ノーマルモードで、 .. parsed-literal:: :class: console Ctrl-w o を入力するたびに、 .. parsed-literal:: :class: program 複数のウィンドウを表示 ←→ 単一のウィンドウを表示 と、ウィンドウの表示状態が切り替わります。 .. image:: /dist/img/vim/zoomwin_split.jpg :alt: split window 複数のバッファを表示中。 |BR| ↓ |BR| 「Ctrl-w o」を入力。 |BR| ↓ .. image:: /dist/img/vim/zoomwin_zoom.jpg :alt: single window 現在カーソルのあるウィンドウがズームアップ。 |BR| ↓ |BR| 「Ctrl-w o」をもう1度入力。 |BR| ↓ .. image:: /dist/img/vim/zoomwin_split.jpg :alt: split window 元の複数のバッファを表示した状態に。 |BR| ↓ |BR| ズームアップしたいウィンドウに移動して、 |BR| ↓ .. image:: /dist/img/vim/zoomwin_move.jpg :alt: move window ↓ |BR| 「Ctrl-w o」を入力。 |BR| ↓ .. image:: /dist/img/vim/zoomwin_zoom2.jpg :alt: zoom window 移動先のウィンドウがズームアップ。 .. parsed-literal:: :class: program このスクリプトを使用した場合には、 確認 ←→ ウィンドウの移動 ←→ 編集 を繰り返す形になるでしょう。 エディタは主に、 - ファイルを編集する時と、 - ファイルの内容を確認する時に 使用されます。 編集する時はひとつの大きなウィンドウで作業した方が集中できますし、 |BR| 複数のファイルを参照しながら編集する場合は、 複数のバッファを見える状態にしておく方が効率が上がります。 |BR| このスクリプトは、なかなか便利なスクリプトではないでしょうか。 ================================================================================ 注意事項など ================================================================================ 一部のウィンドウマネージャーと、同時に扱うと不具合が出てしまうようです。 ウィンドウマネージャを使用している人は注意してください。 (`minibufexpl.vim`__ など。) __ /articles/vim/minibufexpl_vim.html
