================================================================================ javaid.vim |BR| http://www.fleiner.com/vim/download.html Java言語用Syntaxハイライトファイル。 |BR| Java 2 SDK 1.3 までの基本クラスのクラス名をハイライト表示してくれるスクリプトです。 |BR| 大文字、小文字を区別します。 |BR| このスクリプトは、gvim6.0 に標準では含まれていません。 .. image:: /dist/img/vim/javaid.jpg このスクリプトを使うと、Javaソースが、この様にハイライトされます。
