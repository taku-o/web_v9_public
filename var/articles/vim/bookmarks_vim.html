================================================================================ bookmarks.vim : Menu driven bookmarks for gvim |BR| http://www.vim.org/scripts/script.php?script_id=1368 行数指定付きでファイルをブックマークする機能を提供するスクリプトです。 |BR| 記録したブックマークからは、当然、 ブックマークした時の行数目を表示するようにファイルを開くことができます。 ブックマークには、ブックマークされたファイル名、行数の他に、 その行のテキストの一部を表示するので、 |BR| そのブックマークが、ファイルのどの場所をブックマークしたものであるか、 一見でわかるようになっています。 .. image:: /dist/img/vim/bookmarks_vim_menu.jpg :alt: ブックマークには、その行のテキストの一部が表示される。 ブックマークは1つのファイルに対して、いくつでも作成可能です。 このスクリプトはGUI版vim用です。 ================================================================================ インストール方法 ================================================================================ `スクリプトをダウンロード`__ して、 |BR| pluginディレクトリにコピーしてください。 __ http://www.vim.org/scripts/script.php?script_id=1368 .. csv-table:: :header: "ファイル", "URL" "bookmarks.vim", "http://www.vim.org/scripts/script.php?script_id=1368" ================================================================================ 使用方法 ================================================================================ 基本的な操作は、メニューか、ポップアップから行います。 |BR| ポップアップのメニューは、 Ctrlキー押しながらのマウス右クリックを押すと表示されます。 .. image:: /dist/img/vim/bookmarks_vim_popup.jpg :alt: Ctrlキー押しながらのマウス右クリックで表示されるポップアップメニュー ブックマークを追加する ------------------------------------------------------------------------------- GUI版vimエディタのメニューか、 Ctrlキー押しながらのマウス右クリックで表示されるポップアップメニューで |BR| 「Add Bookmark」を選択してください。 |BR| 開いているファイルの、現在のカーソル位置が ブックマークに登録されます。 ブックマークされたファイルを開く ------------------------------------------------------------------------------- GUI版vimエディタのメニューか、 Ctrlキー押しながらのマウス右クリックで表示されるポップアップメニューで |BR| ブックマークを選択してください。 |BR| 選択したブックマークのファイルが開かれます。 ブックマークを削除する ------------------------------------------------------------------------------- ブックマークを削除する場合は、 |BR| ブックマーク情報の保存してあるファイルを編集して、 ブックマークを削除する作業を行います。 GUI版vimエディタのメニューか、 Ctrlキー押しながらのマウス右クリックで表示されるポップアップメニューから |BR| 「Edit Bookmarks」を選択してください。 ブックマーク情報の保存ファイルがvimエディタで開かれます。 |BR| 不要なブックマークを手動で削除してください。 .. image:: /dist/img/vim/bookmarks_vim_edit_bookmark.jpg :alt: ブックマーク情報の保存ファイル編集 ブックマークの編集が終わったら、 |BR| GUI版vimエディタのメニューか、 Ctrlキー押しながらのマウス右クリックで表示されるポップアップメニューから |BR| 「Update Bookmarks」を選択するか、vimエディタを再起動すると、 削除したブックマークが表示されなくなります。 ================================================================================ 設定 ================================================================================ ポップアップメニューの無効化 ------------------------------------------------------------------------------- Ctrlキーを押しながら、マウスの右クリックを押した時に表示する ブックマークポップアップメニュ－を無効にする場合は、 |BR| 下の設定をvimエディタの設定ファイルに追加してください。 .. parsed-literal:: :class: console :let bm_popup = "off" キーマップ ------------------------------------------------------------------------------- 初期の設定ではメニューか、ポップアップ経由でしか |BR| ブックマークの機能を呼び出せません。 キーマップを設定して、 このブックマークの機能をキー入力のみで呼び出せるようにするには、 |BR| 下のようにvimエディタの設定ファイルで設定してください。 .. parsed-literal:: :class: console :nmap 処理を呼び出したいキー :silent call BookmarkAdd() 下の例のように設定した場合は、F5キーを押した時にブックマークを追加します。 .. parsed-literal:: :class: console **" Add Bookmarksを呼び出すキーマップの設定例** :nmap :silent call BookmarkAdd() ================================================================================ 注意事項など
