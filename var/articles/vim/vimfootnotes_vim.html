.. |yen| unicode:: U+000A5 .. YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ VimFootnotes : Insert various types of footnotes with Vim. |BR| http://www.vim.org/scripts/script.php?script_id=431 ドキュメントへの脚注文の入力をサポートするスクリプトです。 次のようなテキストがあるとして、 .. image:: /dist/img/vim/vimfootnotes.jpg :alt: 脚注入力前のオリジナルのテキスト 指定の箇所にマーカーを挿入、 |BR| ファイルの最後に、コメントを入力できるようにします。 .. image:: /dist/img/vim/vimfootnotes_add2.jpg :alt: 脚注の入力 ================================================================================ インストール方法 ================================================================================ `スクリプト`__ をダウンロードして、 |BR| pluginディレクトリにコピーしてください。 __ http://www.vim.org/scripts/script.php?script_id=431 .. csv-table:: :header: ファイル, URL vimfootnotes.vim, http://www.vim.org/scripts/script.php?script_id=431 ================================================================================ 基本的な使い方 ================================================================================ 入力モードで、 .. parsed-literal:: :class: console f とコマンドを入力すると、脚注の入力のウィンドウが開かれます。 上のコマンドで、開かれたウィンドウは、同じく入力モードで 下のコマンドを実行すると閉じることが出来ます。 .. parsed-literal:: :class: console r .. csv-table:: :header: 操作, モード, 説明 f, 入力モード, 脚注入力用のウィンドウを開く。 r, 入力モード, 脚注入力用のウィンドウを閉じる。 .. note:: は初期の設定では、「 |yen| 」になっていると思います。 つまり、実際には、「 |yen| f 」でコマンドを実行できます。 ================================================================================ 設定 ================================================================================ 「vimfootnotetype」をvimの設定ファイルに設定しておくと、 |BR| 使用される脚注の種類を選択できます。 .. parsed-literal:: :class: program **" 設定例** :let vimfootnotetype = "alpha" .. csv-table:: :header: vimfootnotetypeの値, 使用される脚注の種類 "arabic", "[1] [2] [3] ..." "alpha", "[a] [b] ... [z] [aa] [bb] ... [zz] [a] ..." "Alpha", "[A] [B] ... [Z] [AA] [BB] ... [ZZ] [A] ..." "roman", "[i] [ii] [iii] [iv] [v] ..." "Roman", "[I] [II] [III] [IV] [V] ..." "star", "[\*] [\*\*] [\*\*\*] ..." ================================================================================ その他のコマンド ================================================================================ このスクリプトでは、次のようなコマンドが用意されています。 |BR| どれも、脚注の種類と状態を変更するコマンドです。 .. csv-table:: :header: コマンド, 説明, コマンド実行例 FootnoteNumber, 現在の脚注のカウントを指定の数値に変更します。, :FootnoteNumber 9 FootnoteUndo, 現在の脚注のカウントを1減らします。, :FootnoteUndo FootnoteMeta, 脚注の種類を変更して、カウントもクリアします。, :FootnoteMeta alpha ================================================================================ 注意事項など ================================================================================ 下のコマンドを実行すると、それが脚注入力用のウィンドウであってもなくても、 ウィンドウを閉じてしまいます。 .. parsed-literal:: :class: console r 編集中のウィンドウが間違って閉じられると危険なので、スクリプトを弄って、 「r」のキーマップを外しておいた方が良いと思います。
