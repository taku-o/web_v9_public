================================================================================ vimtips.zip : Display a helpful tip the first time you start Vim each day. |BR| http://www.vim.org/scripts/script.php?script_id=88 その日最初にvimエディタを起動した時に、 |BR| vimオフィシャルサイトに投稿されているVim Tipsを表示するスクリプトです。 ================================================================================ インストール方法 ================================================================================ プラグインのインストール ------------------------------------------------------------------------------- まず、 `配布ファイル`__ をダウンロードしてください。 __ http://www.vim.org/scripts/script.php?script_id=88 ダウンロードしたファイルは圧縮ファイルになっているので |BR| 何らかの解凍ツールを使用して解凍します。 このファイルを解凍すると、以下のように展開されます。 .. parsed-literal:: :class: program **圧縮ファイルの構成** doc/ + +--- totd.txt plugin/ + +--- totd.vim + +--- vimtips/ + +--- vimtips.txt docディレクトリ以下のファイルをローカルのdocディレクトリに、 |BR| pluginディレクトリ以下のファイルをそのままローカルのpluginディレクトリにコピーします。 ここまで作業したら、一度vimエディタを起動して、 |BR| エディタ起動時にウィンドウが分割された状態になるか確認してください。 .. image:: /dist/img/vim/vimtips_initial.jpg :alt: プラグインインストール後の初期画面 Vip Tipsのインストール ------------------------------------------------------------------------------- 次にVimのオフィシャルサイトで配布しているVip Tipsのデータを入手して、 |BR| スクリプトから読める位置にコピーします。 Vip Tipsのデータは、 `オフィシャルサイトのこのページ`__ からダウンロードできます。 __ http://www.vim.org/tips/ .. image:: /dist/img/vim/vimtips_download.jpg :alt: ダウンロード画面へのリンクの場所 ファイルサイズは2006年2月13日現在で1.2MBほどあります。 .. note:: 環境によってはダウンロードしづらいかもしれません。 2006年2月13日時点のVim Tipsのデータファイルをこちらに置いておきます。 解凍ツールで解凍してから利用してください。中身はテキストファイルです。 `Vim Tipsデータ`__ __ http://indefero.taku-o.net/index.php/p/download/source/file/master/vimtips20060213.zip ダウンロードしたテキストファイルで、 |BR| 先ほどローカルのpluginディレクトリ内に作成した |BR| vimtipsディレクトリ内のvimtips.txtファイルを上書きしてください。 .. csv-table:: :header: "ファイル", "URL" vimtips.zip, http://www.vim.org/scripts/script.php?script_id=88 vimtips.txt, http://www.vim.org/tips/ ================================================================================ 使用方法 ================================================================================ 日付が変わって、最初にvimエディタを起動した時にVim Tipsが表示されるようになります。 |BR| 表示されるVim Tipsは、前回表示したTipsの次のTipsです。 日付が変わるまで待つ以外に、 下のコマンドをvimエディタで入力して、直接呼び出すこともできます。 .. parsed-literal:: :class: console :TipOfTheDay ================================================================================ 注意事項など ================================================================================ このスクリプトは、Vim Tipsをどこまで表示したかの情報を |BR| 自身のスクリプト内で保持しているので、 |BR| Vim Tipsをまた最初から見る場合は、 |BR| totd.vimスクリプトを最新のファイルと入れ替えてください。
