YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ shellmenu.vim |BR| ftp://ftp.vim.org/pub/vim/runtime/macros/ このスクリプトはVim6.2の配布に含まれています。 |BR| shellスクリプト作成時によく使用される構文が登録されたメニューを vimエディタに追加します。 .. image:: /dist/img/vim/shellmenu.jpg :alt: shellmenu.vimにより作成されるメニュー このスクリプトはGUI版vimエディタ専用スクリプトです。 ================================================================================ インストール方法 ================================================================================ shellmenu.vimは、Vim6.2の配布ソースに含まれていますので、 新たにスクリプトを入手する必要はありません。 下のコマンドを実行するか、_gvimrc設定ファイルに書き込むと、 |BR| shellスクリプト作成用メニューは有効になります。 .. parsed-literal:: :class: console :source $VIMRUNTIME/macro/shellmenu.vim このスクリプトで作成されたメニューを消すには、 下のコマンドを実行してください。 .. parsed-literal:: :class: console :source $VIMRUNTIME/dellmenu.vim :source $VIMRUNTIME/menu.vim このスクリプトで作成されるメニューは幅を取るので、 shellスクリプトを編集する時のみ有効にした方が良いでしょう。 ================================================================================ 使用方 ================================================================================ このメニューは、入力モードの時のみ選択可能になります。 |BR| 挿入したい構文をメニューから選ぶと、 その文がカーソルのある場所に挿入されます。 .. parsed-literal:: :class: program **# 例、Stmts → forを選択した場合に挿入される構文** for in do done ================================================================================ 注意事項など ================================================================================ 画面の解像度が小さいと、メニューの文字が切れてしまいます。
