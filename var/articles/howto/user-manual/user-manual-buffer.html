YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ バッファ ================================================================================ vimエディタは、複数のファイルを同時に編集できるエディタです。 あるファイルを編集している時に、別のファイルを開いて編集できます。 |BR| 新しいファイルを開くと、そのファイルはバッファのリストとしてvimエディタで管理されます。 |BR| vimエディタで開いているバッファを切替えるには、コマンドラインモードで :raw-html:`:buffer` コマンドを使用します。 .. parsed-literal:: :class: console **" バッファを切替える。** :b[uffer] {バッファ} .. csv-table:: :header: コマンド, 説明 :b[uffer] {バッファ}, vimエディタで開いているバッファを切替える .. note:: コマンドラインモードからノーマルモードに戻るには :raw-html:`ESC` キーを、 |BR| ノーマルモードからコマンドラインモードにモードを変更するには :raw-html:`:` キーを |BR| 押します。 ================================================================================ ウィンドウ ================================================================================ ウィンドウを開く ------------------------------------------------------------------------------- vimエディタではウィンドウを複数に分割して、それぞれのウィンドウで別々の編集作業を 行うことも出来ます。 .. image:: /dist/img/howto/user-manual/window.jpg :alt: ウィンドウの分割 ウィンドウを上下に分割するには、コマンドラインモードで :raw-html:`:split` コマンドを |BR| 左右に分割するには、コマンドラインモードで :raw-html:`:vsplit` コマンドを実行します。 .. csv-table:: :header: コマンド, 説明 :sp[lit], ウィンドウを上下に分割する :vs[plit], ウィンドウを左右に分割する .. image:: /dist/img/howto/user-manual/window_sp.jpg :alt: ウィンドウを上下で分割 ↑ :raw-html:`:split` コマンドで上下に分割、↓ :raw-html:`:vsplit` コマンドで左右に分割 .. image:: /dist/img/howto/user-manual/window_vsp.jpg :alt: ウィンドウを左右で分割 ウィンドウを閉じる ------------------------------------------------------------------------------- 分割したウィンドウの一つを閉じるには、その閉じたいウィンドウにカーソルを移動し、 |BR| コマンドラインモードで :raw-html:`:close` コマンドを実行します。 .. csv-table:: :header: コマンド, 説明 :clo[se], ウィンドウを閉じる ================================================================================ タブ ================================================================================ タブを開く ------------------------------------------------------------------------------- vimエディタは、タブエディタとしても使用できます。 .. image:: /dist/img/howto/user-manual/tab.jpg :alt: vimエディタのタブ vimエディタで新しくタブを開くには、 |BR| コマンドラインモードで :raw-html:`:tabedit` コマンドを実行します。 .. csv-table:: :header: コマンド, 説明 :tabe[dit], 新しくタブを開く タブを閉じる ------------------------------------------------------------------------------- 開いたタブを閉じるには、そのタブに移動して、 |BR| コマンドラインモードで :raw-html:`:tabclose` コマンドを実行します。 .. csv-table:: :header: コマンド, 説明 :tabc[lose],
