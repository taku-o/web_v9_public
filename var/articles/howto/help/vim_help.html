YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ vimエディタには、かなり充実したオンラインドキュメントが添付されています。 |BR| vimエディタの操作・使い方が分からなくなった時に、 このドキュメントを検索して、操作・使い方を調べることができます。 ================================================================================ 「help」コマンドの使い方 ================================================================================ 「help」コマンドの書式 ------------------------------------------------------------------------------- 分からない事が出てきた時は、 ノーマルモードで、次の書式のコマンドを実行すると、 |BR| vimエディタはオンラインドキュメントを、その検索ワードで検索し、 |BR| 検索にヒットしたドキュメントを表示します。 .. parsed-literal:: :class: console :help 分からないこと .. image:: /dist/img/howto/help/input_help.jpg :alt: 「help」コマンドの入力例 「help」コマンドの実行例 ------------------------------------------------------------------------------- 例えば、 |BR| 「vsplit」とはどのようなコマンドか？ |BR| と思ったのならば、 |BR| 次のようにコマンドを実行してみてください。 .. parsed-literal:: :class: console :help vsplit .. image:: /dist/img/howto/help/input_vsplit_help.jpg :alt: 「vsplit」コマンドのヘルプの検索 ヘルプを表示する時に、ウィンドウを分割しない ------------------------------------------------------------------------------- 単純にhelpコマンドを実行するとウィンドウが分割され、 新しいウィンドウでヘルプが表示されますが、 |BR| 次のように「only」をつけてコマンドを実行すると、 ウィンドウを分割せず、現在のウィンドウを置き換えてヘルプを表示できます。 .. parsed-literal:: :class: console :help 分からないこと | only ================================================================================ 覚えておくと良いオンラインドキュメントのページ ================================================================================ オンラインドキュメント最初のページ ------------------------------------------------------------------------------- vimエディタに添付されているオンラインドキュメントの最初のページです。 |BR| 上から順にドキュメントを読めば、全てのページを読めるという寸法です。 |BR| このページを表示するには :raw-html:`:help` コマンドを引数無しで実行してください。 .. parsed-literal:: :class: console :help vsplit ユーザマニュアル、How-toリンク ------------------------------------------------------------------------------- - ユーザマニュアル - How-toリンク これら2種のドキュメントは、 vimエディタの操作に関するドキュメントへのリンクを集めたページです。 |BR| vimエディタで、ある操作が分からなくなったけれど、 検索ワードが分からない場合などは、これらのページを読むと解決するかもしれません。 |BR| ユーザマニュアル、How-toリンクを表示するには、それぞれ次のコマンドを実行してください。 .. parsed-literal:: :class: console **" ユーザマニュアル** :help user-manual .. parsed-literal:: :class: console **" How-toリンク** :help howto .. note:: GUI版のvimエディタなら、メニューからでも、これらのページを表示できます。 |BR| vimエディタのメニューバーのヘルプを見てください。 コマンド一覧表ページ ------------------------------------------------------------------------------- 次のコマンドは、vimエディタの各モードのコマンドの一覧表ページを表示します。 |BR| vimエディタの場合、 vimエディタ特有の機能なので名前が分からなかったり、コマンドが短すぎたりで、 |BR| オンラインドキュメントを検索するのも困難な場合がありますが、 |BR| このページを見ると、大抵解決します。 .. parsed-literal:: :class: console :help index .. image:: /dist/img/howto/help/viusage.jpg :alt: ノーマルモードコマンド一覧の表示 ノーマルモードのコマンド一覧を表示する :raw-html:`:viusage` 、 |BR| EXモードのコマンド一覧を表示する :raw-html:`:exusage` というショートカットも用意されています。 |BR| それらのコマンド一覧ページを表示するには、次のコマンドを入力してください。 .. parsed-literal:: :class: console **" ノーマルモードのコマンド一覧の表示** :viusage .. parsed-literal:: :class: console **" EXモードのコマンド一覧の表示** :exusage
