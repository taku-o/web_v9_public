YEN SIGN .. role:: raw-html(raw) :format: html .. contents:: ================================================================================ 概要 ================================================================================ vimエディタを使用して、2つのテキストファイルの差分を表示する方法を紹介します。 |BR| このテクニックは、主にファイルの比較をする場合に使用します。 ================================================================================ 指定のファイルとの差分のとり方 ================================================================================ 「diffsplit」コマンドを使う ------------------------------------------------------------------------------- vimエディタでは「diffsplit」コマンドで、 今エディタで開いているファイルと、指定したファイルとの差分を表示できます。 |BR| 「diffsplit」を使ってファイルの差分を取る場合、コマンドの書式は次のようになります。 .. parsed-literal:: :class: console :vertical diffsplit 差分を取りたいファイル 差分を取りたいファイルの名前が「theother.txt」なら、コマンド例は次の通りです。 .. parsed-literal:: :class: console :vertical diffsplit theother.txt .. image:: /dist/img/howto/diff/vertical_diff.jpg :alt: 2つのテキストファイルの差分表示（横並べ） .. note:: 「vertical」コマンドは「ウィンドウを横2つに分割しろ」という指定です。 |BR| このオプションを外した場合、ウィンドウは縦2つに分割された後、 各差分がそれらのウィンドウに表示されます。 .. parsed-literal:: :class: console **" 差分を取りたいファイルが「theother.txt」の場合のコマンド例** :diffsplit theother.txt .. image:: /dist/img/howto/diff/horizontal_diff.jpg :alt: 2つのテキストファイルの差分表示（縦並べ） 「VDSplit」コマンドを使う ------------------------------------------------------------------------------- `Kaoriya`__ 版パッチのあたっているvimエディタなら、 「vertical diffsplit」と同様の機能を持つコマンドとして、 「VDSplit」というコマンドが組み込まれています。 |BR| この「VDSplit」コマンドを使用すると、 より少ないキーストロークで2つのファイルの差分を表示できます。 __ http://www.kaoriya.net/ .. parsed-literal:: :class: console :VDSplit 差分を取りたいファイル **" 差分を取りたいファイルが「theother.txt」の場合のコマンド例** :VDSplit theother.txt .. image:: /dist/img/howto/diff/vertical_diff.jpg :alt: 2つのテキストファイルの差分表示（横並べ） .. note:: 「VDSplit」コマンドは `Kaoriya`__ 版パッチのあたっているvimエディタでのみ、 使用できるコマンドです。 __ http://www.kaoriya.net/ ================================================================================ 指定のバッファとの差分のとり方 ================================================================================ 差分を取りたい2つ目のファイルがバッファに入っている時（既にエディタで読み込んでいる時）、 |BR| そのバッファは「# + バッファの番号」という書式で指定できるので、 次のような方法でも差分をとることができます。 |BR| この方法は大抵、ファイル名を指定した場合よりもキーストロークが少なくて済むので、 覚えておくと役に立ちます。 .. parsed-literal:: :class: console :VDSplit 差分を取りたいバッファ **" 差分を取りたいバッファの番号が5の場合のコマンド例** :VDSplit #5 .. image:: /dist/img/howto/diff/vertical_diff.jpg :alt: 2つのテキストファイルの差分表示（横並べ） .. note:: バッファの番号を知るには「buffers」コマンドか、「ls」コマンドを実行してください。 |BR| vimエディタのコマンドの中で「# + バッファの番号」と記述すると、 その番号のバッファを指定したことになります。 .. parsed-literal:: :class: console :buffers .. image:: /dist/img/howto/diff/buffers_cmd.jpg :alt: 「buffers」コマンドの実行結果
